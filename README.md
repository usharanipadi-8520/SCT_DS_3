# 🌳 Decision Tree Classifier on Bank Marketing Dataset

## 📌 Project Overview

This project is part of my Data Science Internship at SkillCraft Technology.

The objective of this task is to build a Decision Tree Classifier to predict whether a customer will subscribe to a bank term deposit based on demographic, financial, and marketing campaign information.



## 📂 Dataset

- **Dataset:** Bank Marketing Dataset
- **Source:** UCI Machine Learning Repository
- **Target Variable:** `y`
  - **yes** → Customer subscribed to the term deposit
  - **no** → Customer did not subscribe

## 🎯 Objective

Develop a machine learning model that predicts customer subscription to a term deposit using the Decision Tree Classification algorithm.

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Exploratory Data Analysis (EDA)

The following visualizations were created to better understand the dataset:

- Subscription Distribution
- Age Distribution
- Job vs Subscription
- Correlation Heatmap
- Feature Importance
- Decision Tree Visualization
- Confusion Matrix


## 📈 Results

The Decision Tree model successfully classified customers based on their likelihood of subscribing to a term deposit.

Key findings include:

- Customer demographics influence subscription decisions.
- Previous marketing campaign outcomes significantly affect predictions.
- Feature importance highlights the variables that contribute most to the model.
- The Decision Tree provides an interpretable model for understanding customer behavior.



## 🎯 Conclusion

This project demonstrates the complete machine learning workflow, from data preprocessing and exploratory analysis to model building and evaluation. The Decision Tree Classifier effectively predicts customer subscription behavior while providing clear insights into the factors influencing marketing campaign success.
